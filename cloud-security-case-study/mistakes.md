# Common Cloud Security Mistakes

## 1. Public Storage Buckets
Making storage publicly accessible exposes sensitive data.

---

## 2. Overly Permissive Roles
Giving full access instead of limited permissions increases risk.

---

## 3. Open Security Groups
Allowing 0.0.0.0/0 access exposes services to the internet.

---

## 4. Hardcoded Secrets
Storing API keys in code or .env files is unsafe.

---

## 5. No MFA
Accounts without MFA are easy to compromise.

---

## 6. Sharing Credentials
Makes tracking actions impossible and increases risk.

---

## 7. No Updates
Outdated software contains known vulnerabilities.

---

## 8. No Encryption
Sensitive data can be exposed if not encrypted.
